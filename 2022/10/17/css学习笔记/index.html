1.HTML表单标签

表单中除了提交按钮, 其它项目都要有 name属性值 , 否则表单不能把它提交出去.

2.css层叠样式表

CSS属性值中的长度单位

px 像素 图像中的最小元素

em 一个相对单位, 2em表示浏览默认大小的2倍 

% 相对于上一级元素的百分比

CSS属性值中的颜色单位

（1）颜色名称: red, green, blue, yellow, purple, pink, white, black

（2） 十六进制: 可以用#开头的6位16进制表示. 每两位分别用来表示红色,绿色,蓝色 三种自然色

    # AA   BB   CC 
      红   绿    蓝

     2位16进制可以表示 0~255, 即把每种颜色分为256份,来表示颜色的艳丽程度

      如果两位相同,那么可以简写, #AABBCC 可以简写为 #ABC

（3）rgb(红,绿,蓝): 里面可以是0~255的十进制数值

过style标签，在网页上创建嵌入的样式表

<style>
p{font-size: 30px; #字体大小
  font-style: italic; #斜体
  font-weight: bold;  #加粗
  font-family: '隶书';  #字体
  font-variant: small-caps;} #小型大写字母
  font:15px/1.5 微软雅黑； #简写形式，15像素，1.5倍行高
</style>

<style> 
ul li{color:red;}     /* 后代选择器 ul中的所有li 不管是儿子,还是孙子 */
ul>li{color:green;} /* 父子选择器 ul中的li, 要求li是它的子级元素 选中abcdef */ 
ol+li{color:blue;}     /* 相邻选择器 ol后面的同级的li元素 选中 d */ 
ol~li{color:gold;}     /* 兄弟选择器 ol后面的所有li兄弟元素 选中 def */ 



伪类选择器
<style type="text/css">
     a:link{ color: red; }  #选中 默认状态下的a链接要跳转的颜色
     a:visited{ color: pink; }# 选中 访问过的a
     a:hover{ color: purple; }#选中 鼠标放上去变的颜色
     a:active{ color: yellow; }#选中 鼠标点击一瞬间变的颜色
</style>
在链接中使用：     
<body>
 <a href="#">点击跳转</a>
</body>

外链样式：通过link标签，链接到外部样式表到页面中
